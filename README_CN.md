
# PocketAssistant（随身助手 · Android 版）

> **中文说明（文件名英文）** — 本包满足你的 5 点要求：
> 1) 设置中包含 **代理设置**、**测试代理按钮（占位）**、**API Key 使用方式（网关优先 / 手动填）**、**模型选择**；
> 2) **桌面小组件** 半透明样式，显示日期与待办，**按紧急程度红/黄/绿**上色；
> 3) **聊天功能**：像网页版 GPT 一样发消息，自动**整理为代办**，**写入系统日历**并**更新桌面小组件**；
> 4) 所有代码按 **标准目录结构** 放置；
> 5) 含 **GitHub Actions** 工作流，推到 main/master 即自动出 APK。

## 快速开始
1. 用 Android Studio 打开本项目；首次构建会自动下载 Gradle 8.7；
2. 运行 `:app`（需要通知/日历权限）；
3. 设置页：
   - 勾选“**使用网关中的 Key**” → App 不保存本地 Key；
   - 取消勾选 → 填写 **OpenAI API Key**；
   - **Base URL**：可填你的内网网关地址；
   - **模型选择**：下拉选择（可增删）；
   - **代理/网关**：支持 HTTP/HTTPS/SOCKS5 与用户名/密码；
   - “测试代理与 API”暂为占位，后续可一键发起连通检测。
4. 首页“一句话快速记录”可直接**保存**或**保存并智能整理**（演示版=本地规则，后续切 GPT）。
5. **聊天页**：给 GPT 发消息（当前为演示流程），整理为代办→**写系统日历**→**发通知**→**更新小组件**。
6. **桌面小组件**：半透明背景，今日日期，最多 3 条待办，颜色=紧急程度（红/黄/绿）。

## CI（GitHub Actions）
- 文件：`.github/workflows/android.yml`
- 推送到 `main` 或 `master` → 自动执行 `:app:assembleDebug`，产物在 Actions 的 Artifacts。
